# Шаги для деплоя через Vercel CLI

## Шаг 1: Авторизация

Откройте терминал и выполните:
```bash
vercel login
```

Это откроет браузер для авторизации. После успешной авторизации вернитесь в терминал.

## Шаг 2: Привязка к проекту

После авторизации выполните:
```bash
vercel link
```

Вам нужно будет:
1. Выбрать вашу команду (Scope)
2. Выбрать существующий проект из списка или ввести название

## Шаг 3: Деплой

```bash
# Сначала соберите проект
npm run build

# Деплой в production
vercel --prod
```

---

## Альтернатива: Использование токена

Если хотите использовать токен напрямую (без интерактивной авторизации):

1. Получите токен на https://vercel.com/account/tokens
2. Используйте:
```bash
vercel --token=YOUR_TOKEN --prod
```

---

## Или используйте автоматический деплой через GitHub

Если репозиторий уже подключен к Vercel, просто:
```bash
git push
```
